# Вход в приложение

## Страница логина

Форма ввода почты и пароля, кнопка "Войти", кнопка "Забыли пароль?", редирект на страницу регистрации ("У меня нет аккаунта")

## Страница регистрации

Форма ввода почты, кнопка "Получить код", редирект на страницу логина ("У меня уже есть аккаунт")

## Страница ввода JWT-кода

Форма ввода кода, кнопка "Отправить", кнопка "Отправить снова" спустя время.

## Страница ввода пароля

Форма ввода пароля (поля "Пароль", "Повторите пароль"), кнопка "Сохранить".

# Приложение

## Страница наборов

Список всех наборов пользователя в виде карточек, поиск по названию, фильтр (все, публичные, приватные), кнопка "Создать набор", кнопка профиля, информация о количестве наборов.

## Модальное окно создания набора

Форма создания набора (поля "название", "описание" (опц.), "приватный/публичный"), кнопка "Создать".

## Страница набора

Информация о наборе (название, описание (опц.), количество объектов, режим доступа, дата создания, дата изменения), список объектов (картинки с полями и значениями), кнопки "Добавить объект", "Импортировать данные", "Редактировать" (набор), "Карточки", "Тест", "Экзамен", "Статистика".

## Страница добавления объекта

Загрузка изображения, настройка полей (выбор существующих или загрузка нового) и установка их значений, кнопки "Добавить", "Удалить".

## Страница редактирования объекта

Загрузка изображения (нового, замена), настройка полей (изменение существующих), установка новых значений полей, кнопки "Сохранить", "Удалить".

## Страница импорта данных

Кнопки "Импорт с помощью .csv/.xlsx файлов", "Импорт с ручным заполнением полей".

## Страница ручного импорта

Загрузка массива изображений, отображение изображений в виде карточек, определение полей в этих карточка (из уже существующих или определение новых, новое поле добавляется в существующие), установка значений в этих полях, кнопка "Сохранить".

## Страница импорта с помощью таблицы

Кнопка с пояснением, как это работает, загрузка массива изображений, загрузка .csv/.xlsx файла, кнопка "Сопоставить", отображение изображений и полей в виде карточек.

## Страница редактирования набора

Редактирование полей: название, описание, доступа (приватный/публичный), кнопка "Удалить набор", если доступ публичный, то появляются следующие настройки доступа: доступ к карточкам, к тестам, к экзамену, к просмотру ответов, требуется ли авторизация для прохождения, возможность скопировать ссылку

## Страница "Режимы"

Выбор существующих режимов: "Сетка", "Карточки", "" - по умолчанию стоит режим "Сетка", можно перейти на другие режимы с помощью меню навигации.

## Страница настроек режимов

Указание различных параметров для каждого режима: для карточек - поля, которые отображать на развороте карточки, для тестов - по каким полям будут доступны тесты, ...

## Страница режима сетки

Просмотр всех карточек изображений с полями в виде вертикальной сетки.

## Страница режима карточек

Последовательные просмотр карточек изображений с отслеживанием прогресса, кнопки "Помню", "Забыл".

## Страница тестов

Карточки для тестов, разделенные по определенным полям

## Страница теста

На один экран один тестовый вопрос (картинка, варианты ответа), вопросов столько - сколько всего объектов с этим полем, отслеживания прогресса, прогресс теста сохраняется даже при выходе из него, в конце - экран результатов.

## Страница экзамена

Перед началом: количество вопросов, какие поля будут спрашиваться (можно редактировать), кнопка "Начать". После начала: тестовые вопросы, в конце - экран результатов.

## Страница результатов

Для карточек: количество пройденный карточек, количество "запомненных" и "забытых" карточек, кнопка "Начать заново". Для теста и экзамена: процент правильных ответов, список ошибок, кнопка "Начать заново".

## Страница статистики набора

Информация про конкретный набор о прохождениях различных режимов (приватных и публичных).

## Страница профиля

Личная информация, информация о наборах, кнопки "Изменить пароль", "Выйти".

## Страница изменения пароля

Поля "Старый пароль", "Новый пароль", "Повторить новый пароль", кнопка "Сохранить".
