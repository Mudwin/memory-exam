# Кратко об идее

Платформа для создания учебных наборов, основанных на изображениях и связанных с ними данных. Пользователь загружает изображения, задает набор полей для описания объектов и заполняет их значениями. После этого система автоматически генерирует различные режимы обучения и проверки знаний.

Проект не привязан к конкретной предметной области и может использоваться для изучения различных дисциплин, таких как живопись, архитектура, анатомия, география и т.д.

**MVP**: Презагруженный набор изображений, готовы `json`-файл с предустановленными полями и содержанием, 2 режима - карточки и тест, без отслеживания прогресса.

**Далее**: Загрузка изображений через форму, определение полей и их содержания, система на основе этих данных сама генерирует задания и тесты, пользователь может отслеживать свой прогресс.
